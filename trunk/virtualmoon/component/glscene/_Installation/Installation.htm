Installation of GLScene
into Lazarus LCL

The Instruction for installing GLScene library into Lazarus to work with Free Pascal Compiler (FPC) includes next steps:

1. Download the last release of GLScene_LCL for Lazarus from the page http://sourceforge.net/projects/glscene/files/ as archive file.  You may also to check out source codes of current SVN version of GLScene from the URL of repository https://svn.code.sf.net/p/glscene/code/branch.

 

2. For SVN version make a copy of the trunk in a separate directory, e.g. in the working directory ../GLScene_LCL, to prevent original source codes from occasional changes. You may skip this step if you don’t need to update the code from SVN repository further.

 

3. Copy auxiliary dynamic libraries from ..GLScene_LCL/Source/external into your system directory to support 3D sounds, shaders and physics (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Otherwise you will need to have ones for every application directory with exe file that calls an appropriate dll.

 

4. Setup Library Paths in Options dialog. Add the directory ../GLScene_LCL/Source to search files for GLScene’s applications and samples

 

 

5. Open the lpk packages in your ../GLScene_LCL/Packages directory, compile and install ones.

Note: SDL package should not be installed on Windows 7/8/8.1 due to conflict with Lazarus environment

 

After installation you can start to work with GLScene’s component library and use demos for LCL in “..GLScene_LCL/Samples/Lazarus” directory.
