 
This version is to fix blank display on Mac with ATI graphic card.
It is based on 6.0 with the required change to support the old compiler and library.

It include an old Glscene version from  
http://svn.freepascal.org/svn/fpcprojects/glscene/trunk
at revision 2340

It need to be compiled with fpc 2.4.0 from
http://svn.freepascal.org/svn/fpc/tags/release_2_4_0

And Lazarus 0.9.28.2 from
http://svn.freepascal.org/svn/lazarus/tags/lazarus_0_9_28_2
